2048 - A small game written in C

I've developed a simple 2048 game using C, designed for play within the terminal environment. Players can manipulate the tiles using the 'WSAD' keys, earning points each time they successfully merge tiles. The game also keeps a record of the highest score achieved. What's more, players have the flexibility to pause and resume the game at their convenience. Upon reaching the end of a game session, players are presented with the option to either start a new game or exit the program. The final outcome of the game can result in one of two possibilities: "Lose" or "Win," contingent upon the player's performance.
